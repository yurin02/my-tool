# my-tool

日々の作業を自動化して生産性を高めるためのツールです。シンプルな操作で、複雑なタスクをこなせます。

## 特徴

- **高速処理** — 最適化されたエンジンにより、従来の10倍のスピードでタスクを処理
- **セキュア** — データはすべて暗号化され、安全に管理
- **シンプル操作** — 直感的なUIで、誰でもすぐに使い始められる

## ファイル構成

```
my-tool/
└── index.html   # ランディングページ
```

## 使い方

`index.html` をブラウザで開くとランディングページを確認できます。
